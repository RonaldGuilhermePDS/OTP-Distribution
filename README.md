## Getting Started

Follow the instructions to start the project without any problems.

## Installation

1. Clone the Repository
```sh
git clone https://github.com/RonaldGuilhermePDS/OTP-Distribution.git
```
2. Run with docker compose
```sh
docker-compose up
```
